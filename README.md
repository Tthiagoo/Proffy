## Instalação das dependências :wrench:
<p>Execute esse comando em seu terminal na pasta do projeto</p>

```sh
yarn install 
```

## Mobile :iphone:
<p>Baixe aqui a versão mobile: https://www.mediafire.com/file/r29noe53l2n11b6/proffy.apk/file </p>
